Scratch/Snap! to Csharp Converter



What is this Project:

This project converts Scratch , and Snap! Projects to Avalonia Projects (C#)

What is the Story about this Project:

Well it is pretty simple, i as a Developer saw Projects like Turbowarp and I was interested into the Topic

Originally it started as a Windows Forms Project

You have to install this to make the project work

On Linux: you have to start Installprerequesitaries.sh with the Shell

On Windows: you just have to start the Installprerequesitaries.bat

On MacOS: you just have to start the Installprerequesitaries.sh, but i am not sure if this works

Both install Dotnet and Avalonia.Templates automatically, and fast!

For Source Compilation:

On Linux: use the CompileLinux.sh

On Windows: use the CompileWindows.bat

On MacOS: use the CompileMacOS.sh
